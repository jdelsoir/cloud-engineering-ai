# Cloud Engineering Faction — Gen AI Initiatives

A single-file HTML/CSS presentation showcasing the Cloud Engineering Faction's
initiatives in the Orange Business Gen AI initiative.

Self-contained — no build step, no framework. Open `index.html` in a browser,
or view the live site (below). Navigate with arrow keys, the on-screen arrows,
the dot strip, or the particle-globe hub on slide 1 (each bright node links to
an initiative).

Built to the Orange / Orange Business brand. Every 3D effect is pure HTML & CSS;
the only runtime fetch is the Orange logo over HTTPS.
